mtk.ga.ga_tools module¶

Defines the GATools.

class mtk.ga.ga_tools.GATools(selection, crossover, mutation, normalization, optimization, fitness, exit_, parallel, ga_input)[source]¶

Bases: object

Stores objects which carry out GA operations on populations.

selection¶

Selection – The Selection instance which performs selections of a population’s members.

crossover¶

Crossover – The Crossover instance which crosses a population’s members.

mutation¶

Mutation – The Mutation instance which mutates a population’s members

normalization¶

Normalization – A Normalization instance which rescales or normalizes the fitness values in the population.

optimization¶

FuncionData – Holds the name and arguments of an optimization function defined in optimization. The function is used to optimize the molecules in the population.

fitness¶

FunctionData – Holds the name and arguments of the function in fitness to be used for calculating the fitness of MacroMolecule instances in the population.

exit¶

Exit – An Exit object which checks if the population satisfied the exit criterion to stop the GA early.

parallel¶

bool – If True optimizations and fitness calculations are done on each molecule in parallel.

input¶

GAInput – The GAInput object holding data gathered from the input file.
